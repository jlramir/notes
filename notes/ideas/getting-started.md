# Welcome to Your Notes

This is your **zero-dependency** note-taking system.

## What You Can Do

- Write notes in Markdown
- Organize by folder and tags
- Search from the terminal or the browser
- Switch between 6 game-inspired themes

## CLI Quick Reference

```bash
python main.py new "Note Title" --folder work --tags python,tips
python main.py list
python main.py search query
python main.py build
python main.py theme doom
```

## Themes

Pick the one that fits your mood:

- **Cyberpunk 2077** — neon cyan on near-black
- **The Last of Us** — warm amber, worn serif
- **Red Dead Redemption 2** — western sepia parchment
- **Returnal** — bioluminescent purple/green sci-fi
- **Dead Space** — blood red horror industrial
- **Doom** — brutal orange metal

Open `output/index.html` in your browser after running `build`.
